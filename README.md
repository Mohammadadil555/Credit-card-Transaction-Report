This project is a Credit Card Transaction Analysis Dashboard developed using Power BI. It provides a comprehensive overview of customer transactions, revenue generation, and spending behavior across different segments. The dashboard highlights key metrics such as total revenue, transaction volume, interest earned, and customer demographics through interactive visualizations.

It enables users to analyze trends by quarter, card category, expense type, education level, and customer job profile, helping to identify high-value customers and major spending patterns.
